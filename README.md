# Moradi Portfolio

A concise, static portfolio site showcasing three featured projects, screenshots, and links to source repositories. The site is intended for quick review by hiring managers and engineers.

## Live Site
[https://moradi1412.github.io/Moradi-PortFolio](https://moradi1412.github.io/Moradi-PortFolio/)

## Tech Stack
- HTML5
- CSS3 (`style.css`, `fanta.css`)
- JavaScript
- Font Awesome (icons via CDN)
- Git | Version control |
- GitHub | Repository hosting |
- GitHub Pages | Website deployment |

## Project Overview
This repository is the presentation layer for Mohammad Javid Moradi's portfolio. The homepage (`index.html`) highlights three projects with a short description, a link to each project's GitHub repository, and a screenshot stored under `public/`.

- Trivia Game — React-based trivia client; integrates Open Trivia DB and Giphy for visuals. Repo: https://github.com/moradi1412/Trivia-Game
- Fitness Tracker — described as a full-stack app (Node.js, Express, MongoDB, React). Repo: https://github.com/moradi1412/Fitness-Tracker
- Task Manager — described as React + Firebase for realtime sync. Repo: https://github.com/moradi1412/Task-Manager

## Run locally
1. Clone the repository:

```bash
git clone https://github.com/moradi1412/Moradi-PortFolio.git
cd Moradi-PortFolio
```

2. Serve the folder with a static server:

```bash
npx serve .
# or
python -m http.server 8000
```

Open the URL printed by the server (e.g., `http://localhost:5000`).
